February 1, 2013: New Team Mega Open Now!

A new team mega "Triple Play" is now open for joining. You'll need to complete a solo game in order to join!

This is a 3-team game; the team with the most worlds wins. We'll also be trying out some new team game code here. It won't affect battles at all - if you attack a teammate you'll still destroy their stuff. But you'll be able to see team rankings in com-net.

Check out the game info page for more info, or to join.
